# intro-elasticity

この文書は弘前大学理工学部地球環境防災学科2年生むけ必修科目である自然防災学の一部として，弾性体力学の初歩の授業を行った際の講義ノートです．弾性体力学の広大な領域のうち，ごくごく初歩の部分を，必要となる数学を含めて丁寧に説明しています．
